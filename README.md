# 🛍️ H&M Clone

A simple H&M-inspired e-commerce website built using HTML, CSS, JavaScript, Node.js, and Express.js.

## 🚀 Features

* Product Listing
* Add to Cart
* Shopping Cart
* Quantity Increase/Decrease
* Remove Items
* Total Price Calculation
* Razorpay Payment Integration (In Progress)
* Admin Orders Page

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js

## 📂 Project Structure

* products.html
* cart.html
* admin.html
* order-success.html
* backend/
* css/
